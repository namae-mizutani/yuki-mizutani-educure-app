# 書店在庫管理システム（開発中）

本リポジトリは、プログラミングスクールの自主制作課題（第8章）の成果物を管理しています。
現在は実装フェーズ（8-5）を中心に開発を進めています。

## 1. プロジェクト概要
個人経営の書店における在庫管理の煩雑さを解消し、販売・在庫・発注をスムーズに連携させるためのシステムです。

## 2. 現在の進捗状況
講師の方への確認用に、カリキュラムの工程順にフォルダを構成しています。

- **8-2_題材決定**: ターゲット（書店）の選定と課題の特定
- **8-3_要件定義**: 必要な機能（在庫一覧、販売登録、検索等）の定義
- **8-4_設計**: データベース設計、ER図、画面遷移図の作成
- **8-5_実装**: Spring Boot / MyBatisを用いたシステム

## 3. 開発環境
- Java 21
- Spring Boot 3.x
- MySQL
- Gradle
- VS Code / Linux (Crostini)

## 4. 今後の実装予定
- 在庫の増減をリアルタイムで反映する処理
- 検索・フィルタ機能
